Lab Sheet-10 - MCA203T - OOPS Using Java
Package name: Labsheet10
JDK: 19+ recommended

How to compile:
  javac Labsheet10/*.java
How to run individual programs:
  java Labsheet10.Task1_InputStreamReaderBufferedReader
  java Labsheet10.Task2_OutputStreamWriterConsole
  ... and so on (class names correspond to file names without .java)

Notes:
- Each task implemented in separate .java file under package Labsheet10.
- Some programs expect command-line arguments; check usage messages printed by each main method.
- Task18 simulates concurrent chunk writes; it's a simulation and appends chunks to a file.
- Task12 writes to shared.log in the working directory.
